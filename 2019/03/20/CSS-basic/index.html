6.float
Float会使元素向左或向右移动，其周围的元素也会重新排列。往往用于图像。

一个浮动元素会尽量向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。
浮动元素之后的元素将围绕它，但浮动元素之前的元素将不会受到影响。

清除float：
因为元素浮动之后，周围的元素会重新排列。为了避免这种情况，使用clear属性。
clear属性指定元素两侧不能出现浮动元素。
